Requisitos para abrir o site (Usar npm install)

-Leaflet
-React Leaflet
-Axios (usado para ligar com a Api do Flask)
((posteriormente será trocado))
-Styled Components
-Framer Motion

Para ligar
Carregue o endereço do projeto em sua máquina,e use o npm start
